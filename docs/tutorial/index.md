# Tutorial Guides

These guides are for **day-to-day use** of the managed terminal/editor environment after setup is finished.

Use them as the entry point for normal workflows in the tools this repo manages.

- [tmux](./tmux.md) — sessions, windows, panes, repo-managed workspaces, and the basic workflow for moving around the managed tmux setup
- [Neovim](./neovim.md) — opening files, core editor motions, plugin-backed workflow, and how the managed config is meant to be used
- [Starship](./starship.md) — what the prompt shows, how to read it, and what to expect from the managed shell prompt
- [Ghostty](./ghostty.md) — using the managed terminal profile, tabs and splits, and the terminal-level behavior this repo configures
